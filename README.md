## Introduction
This guide covers tools that every student should know about.
Each section explains what the tool is, what it is used for and why it is helpful.

## VS Code
Written by: Trevor

VS Code (Visual Studio Code) is a free code editor developed byb microsoft.
It is used for writting and editing code in different programming languages.

### Why VS Code is useful:
- Supports many languages (HTML,CSS,JavaScript,Python, etc.)
- Has extensions to add more features
- Built-in terminal and Git support
- Lightweight and fast

VS Code is great for beginners and professional developers.

## Notion
Written by: Skyller

Notion is a all in one productivity tool used for taking notes, planning and organising tasks.

### Why Notion is useful:
- You can create notes and to-do lists
- Good for group projects
- Can store links images and documents
- works on phone and PCs

Notion helps students stay organised.  

## Canva
Written by: Bann

Canva is an online graphic design tool
It is used to create posters,presentations,flyers and social media designs.

### Why Canva is useful
- Easy to use (drag and drop)
- Many free templates
- No design skill needed
- Good for school presentations

Canva helps students make professional designs quickly.
